二、 借款流程
2.1 本协议成立：乙方按照小贷吧平台的规则，通过在小贷吧平台上对甲方发布的借款需求进行投标时，在投标信息确认框中点击“确认”按钮，并在第三方支付机构完成资金划转手续之时。
2.2 出借资金冻结：乙方在投标信息确认框中点击“确认”按钮，并在第三方支付机构完成资金划转手续后，即视为其已经向丁方发出不可撤销的授权指令，授权丁方委托相应的第三方支付机构在乙方小贷吧平台用户名项下的乙方第三方支付账户中冻结资金，冻结金额等同于乙方投标购买金额（乙方不得要求提取或划转该等款项）。资金冻结将在本协议生效时或本协议确定失效时自动解除。
2.3 出借资金划转：在甲方本次借款金额已全部筹满之日，丁方应指令第三方支付机构将乙方第三方支付账户金额中相应冻结的资金，合计等同于本协议第一条所列的“借款本金数额”的资金，划转至甲方第三方支付账户后即视为借款发放成功，借款利息及相关费用开始计算，同时扣除甲方应支付各项费用。
2.4 本协议生效：本协议自丁方将上述借款本金数额扣除甲方应支付各项费用并划转至甲方在存管银行开设的资金账户，即视为借款发放成功，本借款协议生效。
2.5 乙方特此不可撤销地授权丁方根据本借款协议以及乙方与丁方签署的协议向第三方支付机构发出划款指令，扣划乙方第三方支付账户的借款金额。该等扣划无需另行取得乙方的同意或确认。
2.6 乙方特此不可撤销地授权丁方根据本借款协议以及乙方与丁方签署的协议向甲方收取每月应还借款本息、提前结清补偿金（如有）、对甲方借款到期提醒及违约催收和提起诉讼，并同意丁方可将该等授权内容转委托丙方或第三方，以上转委托无需另行取得乙方的同意或确认。

四、 收费及税费
4.1 丙方及其关联方有权就为本协议借款所提供的服务向甲方收取咨询手续费和服务费，上述费用的收取方式由甲、丙双方另行约定。

4.3 居间服务费包括但不限于软件服务费、客户服务费（借款项目调查费、借款到期催收费等），其中软件服务费占居间服务费总额的60%，客户服务费占居间服务费总额的40%。

五、 本息偿还方式
5.1 甲方必须按照本协议的约定按时、足额偿还对乙方的本金和利息（详见本借款协议第一条中分期还款列表）。
5.2 甲方授权丁方委托其指定的第三方支付机构，按还款计划将应付本息（计算公式见下方“乙方当期应收金额的计算公式”）划转至乙方的第三方支付账户下，划转完毕即视为本期还款发放成功。为此，甲方应保证在每期还款日当日中午12:00前的甲方第三方支付账户余额足以支付当期应付本金和利息，并点击“开启自动还款”按钮。乙方每位出借人当期应收金额的计算公式为：
甲方当月还款计划总金额*该出借人出借金额/甲方借款本金数额。
5.3 乙方（出借人）为多人的，按比例享有债权，即乙方每位出借人应收借款本金金额的计算公式为：甲方偿还借款本金总金额*该出借人出借金额/甲方借款本金数额。甲方部分还款情形下，乙方每位出借人应收债权金额计算公式为：
甲方还款金额/甲方应付的全部款项*该出借人的应收金额。
5.4 还款日不受法定假日或公休日影响，还款日届至必须还款。
5.5 如遇到还款当月无还款日对应日的月份，还款日为应还款当月的最后一日。

6.1 作为担保人，丙方同意为本协议下甲方向乙方负有的借款债务承担代偿责任。代偿范围为：借款本金、利息、逾期罚息、违约金、提前结清补偿金（如有），代偿期间为小额借款发放之日起至小额借款到期之日后2年止。

6.3 丙方履行代偿责任后，乙方在本协议项下的所有权利视为已经得到满足和实现，乙方不得再对甲方提出任何请求或主张；丙方可以向甲方进行追偿，乙方应提供合理及必要的协助。
6.4 丙方怠于履行代偿义务情形下，丁方可委托第三方先行为丙方垫付代偿资金，此等垫付行为一经发生即视为丙方担保责任履行完毕，丙方对丁方委托的第三方承担代偿资金及相应资金成本的返还责任，丁方委托的第三方有权向丙方主张债权。
6.5 丁方委托的第三方先行为丙方垫付代偿资金后，丁方于发生代偿当日17:00通过丁方提供的后台系统向丙方推送《小贷吧代偿记录》。
6.6 丙方承诺并保证，其为甲方在本协议项下借款债务所承担的代偿责任为连带保证责任，保证期间为小额借款发放之日起至小额借款到期之日后2年止。丙方已根据丙方章程履行了相应的内部审批程序。

七、 提前还款
7.1 甲方申请提前还款的，乙方授权丁方有权决定是否接受甲方提前还款的申请。若甲方提前还款申请得到允许的，甲方应向乙方支付应还利息、全部借款本金和提前还款违约金，丙方应协助处理提前结清的相关事项。提前还款的款项划转方式与第五条约定正常还款的方式相同。
7.2 甲方在借款还款期间内可随时申请提前还款。
7.3 甲方提前还款的，应当一次性偿还全部剩余借款应付款项，即甲方提前还款应付款项总额为借款本金、应付利息以及违约金之和。
7.4 甲方在逾期还款时申请提前还款的，则应先支付结清所有逾期款项后方可提前还款。
7.5 甲方提前还款，已支付的利息和相关费用不予退还。
7.6 违约金
  7.6.1 标的借款期限在60天（含）以内的，甲方提前还款无需支付违约金；
  7.6.2 标的借款期限在60天（不含）以上的，甲方提前还款违约金为其借款剩余应付未付利息的50%；
  7.6.3 甲方支付的违约金，其中80%作为投资收益补偿支付给乙方，20%作为项目管理费支付给丁方小贷吧平台。
7.7 应付利息
  7.7.1 标的借款期限在60天（含）以内的，甲方提前还款时，应付利息为借款的全部利息；
  7.7.2 标的借款期限在60天（不含）以上的，甲方提前还款时，应付利息为当期利息，当期利息按当期实际借款天数计算；
  7.7.3 甲方支付的应付利息归乙方所有。
八、逾期还款
8.1 如约定还款日12：00前，乙方的第三方支付账户未能收到甲方划转当期应还本息款项的，约定还款日12：00后视为逾期还款。
8.2 逾期还款的，应按照甲方尚未清偿的全部剩余本金与利息之和，自逾期之日按 0.05 %/日的利率按日计收逾期罚息（“逾期罚息”），直至清偿完毕之日止，逾期罚息不计复利。同时，应按照甲方尚未清偿的全部剩余本金与利息之和，自逾期之日按 0.05 %/日的利率按日计收逾期违约金（“逾期违约金”），逾期违约金不计复利。以上逾期罚息、逾期违约金均由乙方授权委托丁方向甲方收取。逾期罚息由乙方收取，逾期违约金由丁方收取。
8.3 逾期款项中的利息不计利息。
8.4 违约情况下，甲方还款顺序为逾期违约金、实现债权费用、罚息、利息、本金。
九、 违约

  9.2.4 甲方的财务状况出现影响其履约能力的不利变化，且不能及时提供有效补救措施的；
  9.2.5 甲方涉及重大案件或资产被采取了财产保全等强制措施或受到行政处罚、刑事制裁，或出现其他导致其无法正常履行职责的事件；
  9.2.6 其他足以影响甲方生产经营活动和乙方借款安全的重大事件或违约事件。
9.3 若发生9.2款所述情形，乙方有权自行或委托丙方、丁方采取下列任何一项或几项救济措施：

  9.3.3 提前终止本协议；

9.4 乙方保留将甲方违约失信的相关信息在媒体披露的权利。因甲方未还款而带来的调查及诉讼费用等将由甲方承担。

十一、债权转让
